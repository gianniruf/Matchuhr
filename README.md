# Matchuhr
a gameclock for sport. Designed for floorball (swiss unihockey rules)

# NOT READY TO USE YET!!!

## Display
Gametime			4x7-Seg <br>
Goals (Points)		LCD
Penalty-Time		LCD (one for both Teams)
Period				LCD
Timeinformation		RGB

## Usage
### RGB
green		time is running
red			time is stopped
blue		pause
white		Timeout (30s)
orange		Shootout
Pink		End of game

### Buttons
ts0			start/stop
ts1			goals guest
ts2			goals home
ts3			2'-guest
ts4			2'-home
ts5			5'-guest
ts6			5'-home
ts7			start Timeout
